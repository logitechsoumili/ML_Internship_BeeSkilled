## Week 4 – Heart Disease Prediction (Capstone Project)

### Objectives

- Predict the likelihood of heart disease using patient health data
- Perform data preprocessing and feature scaling
- Train a Logistic Regression model
- Evaluate classification performance

### Dataset

- UCI Heart Disease Dataset (Kaggle)

### Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

### Model Performance

| Metric | Score |
|----------|----------|
| Accuracy | 83.15% |
| Precision | 86.79% |
| Recall | 84.40% |
| ROC-AUC | 0.903 |

### Deliverables

- Jupyter Notebook
- Confusion Matrix
- ROC Curve
- Project Presentation Video
